# Launch Checklist

## Week 0-1 Setup

- [ ] Run `scripts/generate_assets.py`.
- [ ] Open `data/CultureRadar_MVP_Workbook.xlsx`.
- [ ] Confirm the 15 selected companies and 3 alternates.
- [ ] Set owner initials in `Company Universe`.
- [ ] Configure Google Alerts from `data/google_alert_queries.csv`.
- [ ] Pick 3 pilot companies: one suspected improving, one suspected deteriorating, one stable comparator.
- [ ] Create a public or semi-public judgment tracker view from `data/judgment_tracker.csv`.

## Week 1-2 Pilot

- [ ] Complete four evidence lines for pilot company 1.
- [ ] Complete four evidence lines for pilot company 2.
- [ ] Complete four evidence lines for pilot company 3.
- [ ] Draft 3 internal culture health cards.
- [ ] Measure total hours spent per card.
- [ ] Decide whether public cadence is one or two cards per week.

## Week 2-4 Public Launch

- [ ] Publish the public about page.
- [ ] Publish the methodology article.
- [ ] Publish the first public card.
- [ ] Add the first public judgment to the tracker.
- [ ] Post English X thread.
- [ ] Post Chinese short thread.
- [ ] Save all source links.

## Week 5-8 Weekly Rhythm

- [ ] Maintain Monday-Friday research cadence.
- [ ] Publish one bilingual issue per week.
- [ ] Log at least one falsifiable judgment per week.
- [ ] Complete 10-15 high-quality external interactions per week.
- [ ] Interview 3-5 readers every two weeks.

## Week 9-12 Validation Loop

- [ ] Complete baseline coverage for 12-15 companies.
- [ ] Audit all open judgments for missing thresholds or deadlines.
- [ ] Close any due judgments honestly.
- [ ] Do not publish hit-rate percentages before 10 closed judgments.
- [ ] Publish the three-month review.
- [ ] Decide next mode: continue free, invitation group, or founder membership.
